\# Binance Futures Testnet Trading Bot



A simple Python CLI app to place MARKET and LIMIT orders on Binance Futures Testnet.



\## Features

\- BUY and SELL support

\- MARKET and LIMIT orders

\- CLI input validation

\- Logging to file

\- Exception handling

\- Direct Binance REST API integration
